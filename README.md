# ☕ Bahr Coffee Store — POS System

Static POS system that saves all data directly to Google Sheets.
Runs on GitHub Pages — no server required.

## Live Site
https://omarelgazzar0.github.io/Bahr-Coffee-Store

## Data
All orders, customers, and catalog items are stored in Google Sheets:
https://docs.google.com/spreadsheets/d/1WpzhVCLM3RIOaOIUt07ttbOtEJHE7lO8NJg6LYpFiAQ

## Export
Click **Export to Excel / Access** on the Customers or Orders page
to download 3 CSV files that open in Excel and import into Access.
